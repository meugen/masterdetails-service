# Service to process Master/Details resources

### Consists of two repositories:
* masterdetails-service - main service implementation **You are here**
* [masterdetails-compose](https://github.com/meugen/masterdetails-compose) - docker compose configuration to service up and running
* [masterdetails-terraform-aws](https://github.com/meugen/masterdetails-terraform-aws) - terraform implementation to apply infrastructure at AWS

### Uses technologies:
* Spring Boot 3.5.7
* Spring Validations
* Spring Data JPA
* Hibernate ORM
* PostgreSQL
* Flyway
* Redis
* MapStruct
* Docker
* Docker Compose
* Terraform
* AWS

### Can be tested at [masterdetails](https://masterdetails.meugen.in.ua/)
